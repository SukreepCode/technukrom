## Techนุกรม
Technology + สารานุกรม = Techนุกรม

Main app of Techนุกรม

Init project
```
npm install -g next now cross-env
```

Dev
```
yarn
yarn dev
```

Firebase things
```
npm install -g firebase-tools
firebase login
firebase use
firebase deploy
```

Init for deploy on now.sh
```
now secret add firebase-api-key [FIREBASE API KEY]
```
